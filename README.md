# sbrx
   
   2D Racing Arena Fighter.
   

   

   
## Usage
	add dependency to your Cargo.toml:
	
	[dependencies]
	sbrx = "0.1.9"	
	
	or:
	>cargo install sbrx = "0.1.9"
	
	then:
	>cargo run
